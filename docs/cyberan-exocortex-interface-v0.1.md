# Cyberan Exocortex Interface Spec v0.1
## Core Principle
Every exocortex-mediated thought must route through ASTRO-CLEAR attestation before influencing:
- Motor/output actions
- Long-term memory encoding
- External world-model updates
- Agent coordination signals
- Irreverisible actions
- Actions that affect the option space of humanity
- Actions under asymmetric agency

## Root Loop (from the StarVasa v0.3 diagram)
observe (orbital) → attest (ASTRO-CLEAR) → reason (AGPI) → authorize → execute (lab/BCI action) → verify → update

## Glyph Attestation at BCI Edge 
- Incoming sensory/thought token → hashed against CRL physics state
- Incoming action/motor output weighed against agency preverservation and mutual co-evolutionary constraints
